
| Specifier | Meaning           |
| --------- | ----------------- |
| `%d`      | signed int        |
| `%u`      | unsigned int      |
| `%f`      | float             |
| `%lf`     | double            |
| `%c`      | char              |
| `%s`      | string            |
| `%p`      | pointer (address) |
| `%x`      | hex (lowercase)   |
| `%X`      | hex (uppercase)   |
